Changes Summary

REMOVED Colored dots on section headers — The blue, green, purple, amber dots next to OVERVIEW, BALANCE SHEET etc. added visual noise without meaning. Clean uppercase labels are enough.
REMOVED Chevron arrows on expandable items — Sub-items are always visible (not collapsible), so the rotating chevrons were misleading. Removed for cleaner look.
CHANGED Net Wealth card — Replaced the separate stats row (with trend icons) with inline dot+label above the ratio bar. Adds a subtle blue gradient background and corner glow for premium feel. More compact, same information.
CHANGED Monthly surplus badge — Now includes a tiny trend arrow icon inside the pill for better visual communication at a glance.
CHANGED Sub-items always expanded — Liabilities sub-items now show by default (matching Assets behavior). No hidden content behind chevrons.
